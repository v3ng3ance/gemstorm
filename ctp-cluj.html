🚌 CTP Cluj · Live Tracker

Ai nevoie de un API key gratuit de la Tranzy (platforma oficială a CTP Cluj).

  1. Mergi la tranzy.dev/accounts/my-apps și creează un cont gratuit.
  2. Apasă „New App", selectează CTP Cluj ca agenție și salvează.
  3. Copiază API Key-ul generat și pastează-l mai jos.

Cheia se salvează în localStorage-ul browserului. Agency ID pentru CTP Cluj = 1 (setat automat). Include liniile din Florești.
